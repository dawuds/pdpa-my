# T4 Checkpoint — produce a sample finding and pass the partner-review gate

> **Tier 4 / Lesson 5 — 120 minutes.** Deliverable: a sample finding on a control of your choice, passing the partner-review gate per Citation Discipline Guide §1 (10-item self-check).

## The exercise

Pick a control. Walk it through to a fully-formed finding. Self-grade against the rubric.

### Choosing a control

Select from the 28 critical-tier controls. Recommended starting controls for first-time T4 graduates (each has substantive material in the issued April 2026 guidelines):

| Control reference | Domain | Why a good starting point |
|---|---|---|
| **DPO-01** | Governance | Universal applicability; clear statutory basis (s12A); operational scope is well-defined |
| **DPIA-01** | DPbD | High-risk processing identification; uses DEICA methodology; clear citation patterns |
| **DPB-04** | DPbD | Children's protection; covers the issued-vs-PCP discipline; most-cited error area |
| **ADM-02** | ADMP | Transparency / §9.3 disclosure; tests the mandatory-vs-permissive distinction |
| **ADM-04** | ADMP | Refusal / s38 anchoring; tests the consent-vs-explicit-consent downgrade |
| **CBT-02** | Cross-Border | Mechanism selection (s129 / s129A); TIA framework |
| **BRR-02** | Breach Response | s12B 72h notification; awareness-timestamp doctrine |

### The execution steps

For your chosen control:

1. **Frame the scenario** — what type of client, what processing activity, what gap
2. **Identify the audit procedure** — which work-program procedure tests this control (DPB-04 step 1; ADM-02 step 3; etc.)
3. **Determine the citation** — which sections / guideline / regulation; ISSUED or LEADING PRACTICE
4. **Verify against the Citation Discipline Guide** — run the 10-item self-check
5. **Write the finding** — 5 Cs structure + all metadata
6. **Self-grade**

## Scoring

| Score | Action |
|---|---|
| 27-30 / 30 | Pass partner-review gate; ready for engagement work |
| 22-26 | Borderline; fix the specific gaps before submitting |
| 17-21 | Material gaps; rework the finding |
| ≤16 | Re-do T4 from the weakest lesson |

## The transition out of T4

A T4 graduate is ready to:

1. **Execute audit procedures** — run a procedure end-to-end with appropriate test methods and sampling
2. **Draft findings** — produce findings that pass the 10-item self-check
3. **Apply citation discipline** — distinguish ISSUED REQUIREMENT from LEADING PRACTICE; cite section numbers; verify against BM source
4. **Self-grade against the rubric** — recognise their own gaps before partner review catches them

T4 graduates still need engagement-manager and partner review at this stage. The objective is to **pass that review on the first attempt**. The previous-baseline data is concerning: 48% of pre-T4 findings had at least one PCP-vs-issued citation error. The target post-T4 is <10%.

## Continuing professional development

Audit practice does not stop at T4. The continuous-improvement items:

| Item | Cadence |
|---|---|
| Read each new JPDP issuance within 2 weeks of release | Reactive |
| Review the QA Reviews archive for methodology updates | Monthly |
| Refresh on the Citation Discipline Guide | Quarterly |
| Refresh on the issued April 2026 BM source PDFs | Annual minimum |
| Cross-jurisdictional updates (GDPR, EDPB, CJEU, ICO, EU AI Act) | Continuous |
| Sectoral CoP updates | When issued |
| Tabletop participation | At least 1 per year |
| Internal audit-quality review participation | Per programme |

The first 12 months post-T4 are still apprenticeship. Use engagement-manager and partner reviews as learning opportunities; capture patterns where your work fell short; close those patterns before the next engagement.

## The path beyond T4

For consultants who want to progress beyond audit execution:

1. **Engagement manager** — leading multi-auditor engagements; partner-review interface; client-facing senior contact
2. **Subject-matter expert** — deep specialisation in a domain (cross-border; ADM/AI; breach response; banking sector; healthcare sector)
3. **Engagement partner / Director** — engagement letter; scope; methodology owner; signing partner for reports

Each of these is a separate development track with its own competency model. The current curriculum (T1-T4) covers the foundation; advanced tracks are bespoke per consultant.

## What's next

T4 is the final tier in the structured curriculum. Beyond T4, the consultant operates under engagement-manager and partner supervision on real engagements.
